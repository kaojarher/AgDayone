# 內建 MCP 協定 · 擷取 Google 搜尋結果 · 無伺服器端依賴
🧩 下載並安裝 MCP Content Fetcher
此擴充套件實作 MCP (Model Context Protocol)，讓 Agent 能透過 Chrome Extension 取得 Google 搜尋結果的頁面內容，繞過 CORS 限制。
1 下載 .zip 檔案後，解壓縮到本機資料夾
2 開啟 Chrome，前往 chrome://extensions/
3 開啟右上角「開發人員模式」
4 點選「載入未封裝項目」，選擇解壓縮後的資料夾
5 擴充套件安裝完成後，回到此頁面即可使用 MCP 功能
擴充套件架構
// manifest.json — MCP Content Fetcher { "manifest_version": 3, "name": "MCP Content Fetcher", "version": "1.0.0", "description": "MCP protocol for AI agent web content fetch", "permissions": ["activeTab", "scripting", "storage"], "host_permissions": ["https://www.google.com/*", "<all_urls>"], "background": { "service_worker": "background.js" }, "content_scripts": [{ "matches": ["<all_urls>"], "js": ["content.js"] }] }
